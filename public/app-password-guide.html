How to get your Gmail App Password

Use a Google App Password instead of your normal Gmail password when the portal asks for App Password. This guide shows the shortest path to create one.

1

Turn on 2-Step Verification

Open your Google Account, go to Security, and enable 2-Step Verification if it is not already on.

2

Open App Passwords

After 2-Step Verification is enabled, search for App passwords in your Google Account security settings.

3

Create a new password

Select the app and device, then generate the password. Copy the 16-character code Google gives you.

4

Paste it into the portal

Return to the Doctor Patient Portal and paste that code into the App Password field.

Important: never use your normal Gmail login password here. Use the generated App Password only.
